
### 项目名称

基于django的视频点播网站开发

### 项目背景

学习完毕python和django之后，想找个练手项目，打算写一个视频点播网站,所以想自己实现一个简单的视频点播网站，学以致用。

### 项目功能
本项目分为前台和后台

前台功能
- 视频列表展示
- 视频播放详情
- 详情评论
- 个人中心

后台功能
- 视频管理
- 评论管理
- 用户管理
- 反馈管理
